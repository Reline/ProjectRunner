# Project Runner
An android game for class

## Dependencies
Apache Ant

64bit JDK 8

Android NDK

Android SDK API 15

## How to build
`ndk-build`
`ant debug install`

## Libraries
SDL2

Google Play Game Services

## Specifications
Procedural endless level

3 lanes

Turning right and left

Random obsticles you must avoid

Leaderboard local and global
